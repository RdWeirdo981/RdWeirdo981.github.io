Notice:

If you have more than one blog under different GitHub accounts, you will find that it is a problem to push your various blogs. I will suggest two ways here:

  1. Do not save your state. Every time you push your blog, you should login again using the right account.

  2. Use different SSH keys to keep your blogs safe. I think this is a safer way, but not necessary. I will write another article to explore what a pair of SSH key is, how it can be used to generate safe connection between website and our localhost, and how it can be used to manage two GH pages.
